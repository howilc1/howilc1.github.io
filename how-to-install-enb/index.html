Steps to Install ENB:

To install ENB, begin by following these steps:

  1. Download the appropriate ENB for your game from the official ENB website. Be sure to select the correct ENB based on the game you will be playing.
  2. Extract the contents of the downloaded ENB to the game’s primary directory.
  3. Next, download and install the required preset located in the ENB file. This step is crucial as the preset will determine the lighting and graphics modifications made to the game.
  4. After downloading the preset, extract it to the primary game directory.
  5. Finally, launch your game and enjoy your newly enhanced gaming experience!

Using ENB with Skyrim:

First, open the Skyrim launcher and go to “options.” From there, select “Advanced” and turn off “Antialiasing” and “Anisotropic Filtering.” Next, open the “SkyrimPrefs.ini” file and set the following values:

  • iPresentInterval=0
  • bTreesReceiveShadows=1
  • bDrawLandShadows=1
  • bFloatPointRenderTarget=1
  • iShadowMapResolutionPrimary=2048
