How to Install Exodus in Kodi

Installing exodus in Kodi is actually quite easy. The first thing you’ll need to do is make sure you have Kodi installed on your device. Then, follow these steps:

  1. Open Kodi and go to the Settings menu.
  2. Select File Manager.
  3. Select Add Source.
  4. Enter the following URL: http://lazykodi.com/
  5. Name the source “lazykodi” and click OK.
  6. Go to the home screen and select Add-ons.
  7. Select the package installer icon.
  8. Select Install from zip file.
  9. Select lazykodi and then select Repositories.
  10. Choose the repository that matches your version of Kodi.
  11. Select Install from repository and choose the lazykodi repository.
  12. Select Video add-ons and then select Exodus.
