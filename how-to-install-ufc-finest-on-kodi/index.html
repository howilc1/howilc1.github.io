Installation Steps for UFC Finest on Kodi

First, you will need to open Kodi and go into the “Settings” option. From there, select “System Settings” and then “Add-ons”. Be sure to turn on “Unknown Sources” in order to download add-ons that are not officially licensed by Kodi.

After turning on “Unknown Sources”, go back to the home screen and select “Settings” again. Then, click on “File Manager” and select “Add Source”. A new window will pop up where you will need to type in the following URL exactly: https://kodiuk.tv/repo/. Name this new source “KodiUKTV”.

Go to the KodiUKTV option under “Add-ons” and select “Install from Zip File”. Select the Kodil.zip file and it will install the new Kodil repository. Go back to the “Add-ons” section and select “Install from Repository”. Choose Kodil Repository and then Video Add-ons. Find UFC Finest and click to install.
