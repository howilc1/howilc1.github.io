Are you looking to add Bob on your Kodi media player? Bob is an incredible Kodi addon that offers a wide range of movies, TV shows, live channels, sports, and more. This blog post will guide you on how to install Bob on Kodi.

Pain Points

Even if you are an experienced Kodi user, getting Bob on your Kodi media player can be a challenge. There are many add-ons with similar names, and it is hard to know which one is the perfect fit for you. Also, Kodi media player is not universally compatible. Hence, users often run into issues when installing third-party addons on the media player.

How to Install Bob on Kodi

Before installing Bob on Kodi, you need to verify if your Kodi settings are set to accept third-party addons. Go to the Kodi settings menu, select System settings, and choose Add-ons. Make sure the “Unknown Sources” option is enabled.

The next step is to add the Bob Kodi addon repository to your media player. Go to the “File Manager” tab in Kodi settings and select “Add source”. Next, enter the path http://dandymedia.club/repo and click “OK” to save it. After that, give it a name and add it to your media source list.

Then, navigate to the “Add-ons” option from the Kodi menu. Click the ”Package installer” icon to install the Bob Kodi repository from the files you added in the previous step. Next, choose “Install from zip file,” select the repo you added, and click “OK”. Wait for a while until you see an enabled notification.

Once the repository is installed, select “Install from repository” and choose “Dandymedia repository.” In the “Video addons” section, click on “Bob Unleashed” and then click “Install” to add the Bob addon to your Kodi media player.
