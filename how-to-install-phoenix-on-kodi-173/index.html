How to Install Phoenix on Kodi 17.3

The process of installing Phoenix on Kodi 17.3 is not as difficult as it seems. Here are the steps you should take to install Phoenix on Kodi 17.3:

  1. Open Kodi 17.3 and go to the System Settings menu.
  2. Click on File Manager and then click on Add Source.
  3. Enter “http://fusion.tvaddons.ag” into the URL field and then enter “Fusion” in the name field.
  4. Go back to the home screen and click on Add-ons.
  5. Click on the Package Installer icon and then select Install from Zip File.
  6. Select the Fusion directory and then click on xbmc-repos.
  7. Choose “english” and then select “repository.xbmchub-3.0.0.zip”.
  8. Wait for the notification message to appear on your screen. Then, click on “Install from Repository”.
  9. Choose the TVADDONS.CO Add-on Repository and then select Video Add-ons.
  10. Scroll down and click on Phoenix.
  11. Finally, click on Install.
