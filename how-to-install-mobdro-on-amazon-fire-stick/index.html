How to install Mobdro on Amazon Fire Stick?

Before we dive into the installation process, note that Mobdro is a third-party streaming application. Therefore, you’ll have to install it manually. Follow the below steps:

  1. First, ensure that your Amazon Fire Stick is set to allow the installation of applications from unknown sources. You can do this by heading to Settings > My Fire TV > Developer Options > Apps from Unknown Sources.
  2. Next, visit the Mobdro website and download the APK file.
  3. Once the download is complete, launch the File Explorer on your Amazon Fire Stick.
  4. Navigate to the location where you downloaded the Mobdro APK file.
  5. Select the file and hit Install to initiate the installation process.
  6. Once the installation is complete, launch Mobdro from Your Apps & Channels.

Congratulations, you have successfully installed Mobdro on your Amazon Fire Stick!
