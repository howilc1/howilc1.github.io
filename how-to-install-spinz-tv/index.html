How to Install Spinz TV

Follow these simple steps to install Spinz TV on Kodi:

  1. Open Kodi and click on the “gear” icon to go to the settings page.
  2. Select “System” and turn on the “Unknown Sources” option.
  3. Click “Yes” on the warning pop-up.
  4. Go back to the Kodi home screen and click on the “gear” icon again.
  5. Select “File Manager” and then click on “Add Source”.
  6. Enter the following URL: http://repo.husham.com/ and click “OK”.
  7. Name the repository “Husham” and click “OK”.
  8. Go back to the Kodi home screen and select “Add-ons”.
  9. Click on the “Box” icon and select “Install from zip file”.
  10. Select the “Husham” repository.
  11. Select “repository.husham.x.x.zip” and wait for the installation to complete.
  12. Select “Install from repository”.
  13. Select “Husham’s repository”.
  14. Select “Program add-ons”.
  15. Select “SpinzTV Wizard” and click “Install”.
  16. Wait for the installation to complete and select “Build Menu”.
  17. Choose the Spinz TV build that you want to install.
  18. Wait for the build to download and install.
  19. Restart Kodi and enjoy streaming with Spinz TV!
