How to Install Navi-X

The first step in installing Navi-X is to download the repository for Kodi. Start by launching Kodi, navigating to the settings tab, and selecting File Manager. From there, click on Add Source and input the following URL: http://fusion.tvaddons.co. Next, select Ok, and give your new source a name. Once you’ve done that, select OK, and your new source will be added to your system.

After adding the repository, navigate to the Add-ons section and click on the Package Installer icon. From there, select Install from Zip File and choose the source you added in the previous step. Next, select the repository.colossus zip file and allow it to install. Once the installation is complete, return to the previous menu and select Install from Repository. Choose Colossus Repository, navigate to the Video Add-ons section, and select Navi-X. Finally, select Install, and Navi-X will be added to your add-ons list.
